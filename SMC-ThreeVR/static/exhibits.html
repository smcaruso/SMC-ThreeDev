Installation Instructions
  • Windows: run the installer and follow its steps. The app will install to Program Files by default and appear in the Start Menu as SMC Portfolio Exhibit. If you need to, the app can be uninstalled through the Add/Remove Programs control panel.
  • Mac: mount the DMG disk image and drag the app icon into your Applications folder. If you’ve set your Mac to only allow App Store applications to run, you may need to give it special permission in System Preferences.
